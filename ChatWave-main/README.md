# 🌊 ChatWave

**ChatWave** is a real-time communication platform built with the **MERN stack** where people can connect, share knowledge, and learn new languages through **text chat** or **video calls**.  
It offers a clean UI and smooth user experience to make online conversations more engaging and productive.

🔗 **Live Demo**: [ChatWave](https://chatwave-1-wqrb.onrender.com)

---

## ✨ Features

- 💬 **Real-time Messaging** — Chat instantly with other users.
- 🎥 **Video Calling** — Connect face-to-face for better communication.
- 🌍 **Language Learning** — Practice and learn different languages from native speakers.
- 📚 **Knowledge Sharing** — Exchange ideas and expertise.
- 🎨 **Modern UI** — Clean, intuitive, and responsive design for all devices.
- ⚡ **Fast & Reliable** — Powered by WebSockets for real-time updates.

---

## 🛠 Tech Stack

**Frontend**:
- React.js
- Redux (for state management)
- Tailwind CSS / Styled Components (modern UI)
- Socket.io-client

**Backend**:
- Node.js
- Express.js
- MongoDB + Mongoose
- Socket.io
- WebRTC (for video calling)
